Our Work Efforts we made during the hack

  1. Home
  2. Our work

Getting location out of photos

One of the things we worked on was pulling the latitude and longitude out of photos with exif data. We achieved this using a python script that we can run from the command line.

Find the area of the photos

We also needed the general area that the photo album came from. In the same python scipt we can draw a square around the album.

Finding height data

In order to 3D print the area, we need to generate an STL of the terrain. We have been using a number of tools, from web apps to the USGS API to find this data.

Generating QR codes

We also need a QR code to link the digital photos to the physical model. We have an automated python command line script that generates a QR code for any string using OpenSCAD.

Merging the QR code and the model

Again, using open SCAD we can merge the extruded QR code layers with the STL of terrain. We can also add pins in the locations of the photos.

3D printing

Once we've assembled all of the pieces we send them to a dual extruder 3D printer to print the QR code.

Output

When done, we leave the user with a physical model to help him remember their trip.
